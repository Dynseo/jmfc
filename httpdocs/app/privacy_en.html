Data Processing

The JeMeFaisComprendre App is an augmentative and alternative communication (AAC) application. The application can be individually adapted to the needs of users. It is possible to add custom images, texts, or audios to the user grids. All these types of data are subsequently called "personal data" or "personalized data".

Offline

All personalized data is stored locally on the user's device. Data is stored in an internal database of the browser (IndexedDB) which is accessible only to the current user of the device.

Online

All personalized data is also stored in a database of Eryn Playground and can therefore be accessed and synchronized on multiple devices of the user. However, all personal data is encrypted end-to-end. This means that only the user himself can see it unencrypted, when logging in with their username and password. Eryn Playground has no possibility to access or process personal data, which are saved in an encrypted manner. The key for encrypting the data is the user-chosen password. For encryption, the AES algorithm and the sjcl software library are used.

The following data can be found in Eryn Playground's database without having the decryption keys:

  • Username and email: the freely chosen username and email are stored unencrypted. Username is only used to identify the user account.
    The username and email address are the only data records that can possibly be linked to a person.
  • Number of grids: although all content is encrypted, it is possible to see the number of grids a user has created.

Access Analysis

For the JeMeFaisComprendre App, we use an access analysis tool from Matomo. All data collected in this way is stored on our own servers, will not be passed to third parties, and cannot be linked to individual persons. The following data is saved:
  • Anonymized IP addresses of the users (the last 2 bytes are deleted, e.g., 89.128.xxx.xxx)
  • Count and duration of access
  • Operating system, browser version, and screen resolution used for access

Legal Basis

The legal basis for the usage of the username and email is consent, as the user has to consent to the data processing when registering.

Data Transfer

If you use synthesized voices with the suffix "online" (see "Settings" -> "User settings" -> "Voice") or if you use the default voice and only voices with the "online" suffix are available, all pronounced texts are sent to the respective online services in order to generate the speech output.
When using voices with the "offline" suffix, no data is sent to external services.

Apart from this, no data is transferred to third parties.

Data Storage

All user data of online users will be deleted after 365 days without use (no login by the user).
If you wish to have your data deleted before this period, please send a request to contact@jemefaiscomprendre.com.
If you wish to delete your data yourself, you can do so directly in the application in Settings > User management > Delete my account.
This will delete your account and all related data.

Revocation and Information on Rights

Consent for data processing can be revoked at any time by contacting contact@jemefaiscomprendre.com. In the event of revocation, the data of the respective user will be deleted.

Further information on your rights (right to information, deletion, etc.) can be found on the CNIL GDPR info page. If you have any questions, you can contact contact@jemefaiscomprendre.com at any time.
