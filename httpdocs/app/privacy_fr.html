Traitement des données

L'application JeMeFaisComprendre est une application de communication améliorée et alternative (CAA). L'application peut être adaptée individuellement aux besoins des utilisateurs. Il est possible d'ajouter des images, textes ou audios personnalisés aux grilles de l'utilisateur. Tous ces types de données sont ensuite appelés "données personnelles" ou "données personnalisées".

Hors ligne

Toutes les données personnalisées sont stockées localement sur l'appareil de l'utilisateur. Les données sont stockées dans une base de données interne du navigateur (IndexedDB) qui est accessible uniquement pour l'utilisateur actuel de l'appareil.

En ligne

Toutes les données personnalisées sont également stockées dans une base de données de Eryn Playground et peuvent donc être accessibles et synchronisées sur plusieurs appareils de l'utilisateur. Cependant, toutes les données personnelles sont chiffrées de bout en bout. Cela signifie que seul l'utilisateur lui-même peut les voir en clair, en se connectant avec son nom d'utilisateur et son mot de passe. Eryn Playground n'a aucune possibilité d'accéder ou de traiter les données personnelles, qui sont enregistrées de manière chiffrée. La clé de chiffrement des données est le mot de passe choisi par l'utilisateur. Pour le chiffrement, l' algorithme AES et la bibliothèque logicielle sjcl sont utilisés.

Seules les données suivantes peuvent être trouvées dans la base de données de Eryn Playground sans avoir les clés de déchiffrement :

  • Nom d'utilisateur et e-mail : le nom d'utilisateur et l'e-mail librement choisis sont stockés en clair. Le nom d'utilisateur est uniquement utilisé pour identifier le compte utilisateur.
    Le nom d'utilisateur et l'adresse e-mail sont les seules données susceptibles d'être liées à une personne.
  • Nombre de grilles : bien que tout le contenu soit chiffré, il est possible de voir le nombre de grilles créées par un utilisateur.

Analyse des accès

Pour l'application JeMeFaisComprendre, nous utilisons un outil d'analyse des accès. Toutes les données collectées de cette manière sont stockées sur nos propres serveurs, ne seront pas transmises à des tiers et ne peuvent pas être liées à des personnes individuelles. Les données suivantes sont enregistrées :
  • Adresses IP anonymisées des utilisateurs (les 2 derniers octets sont supprimés, par exemple 89.128.xxx.xxx)
  • Nombre et durée des accès
  • Système d'exploitation, version du navigateur et résolution de l'écran utilisés pour l'accès

Base légale

La base légale pour l'utilisation du nom d'utilisateur et de l'e-mail est le consentement, lors de l'inscription, l'utilisateur doit consentir au traitement des données.

Transfert des données

Si vous utilisez des voix de synthèses avec le suffixe "online" (voir "Paramètres" -> "Paramètres utilisateur" -> "Voix") ou si vous utilisez la voix par défaut et qu'il n'y a que des voix avec le suffixe "online" disponibles, tous les textes prononcés sont envoyés aux services en ligne respectifs afin de générer la sortie vocale.
Lors de l'utilisation de voix avec le suffixe "offline", aucune donnée n'est envoyée à des services externes.

En dehors de cela, aucune donnée n'est transférée à des tiers.

Stockage des données

Toutes les données des utilisateurs en ligne seront supprimées après 365 jours sans utilisation (aucune connexion de l'utilisateur). Si vous désirez que l'on supprime vos données avant cette échéance, veuillez envoyer une demande à contact@jemefaiscomprendre.com.
Si vous désirez supprimer vous-même vos données, vous pouvez le faire directement dans l'application dans Paramètres > Gestion utilisateur > Supprimer mon compte.
Cela supprimera votre compte et toutes les données qui y sont liées.

Révocation et information sur les droits

Le consentement au traitement des données peut être révoqué à tout moment en contactant contact@jemefaiscomprendre.com. En cas de révocation, les données de l'utilisateur concerné seront supprimées.

De plus amples informations sur vos droits (droit à l'information, à la suppression, etc.) peuvent être trouvées sur la page d'information sur le RGPD de la CNIL. Si vous avez des questions, vous pouvez contacter contact@jemefaiscomprendre.com à tout moment.
